# 美景随享

一个展示世界各地精美风景照片的网页小应用，每次刷新都会从图片服务随机获取新图片，让你在浏览器中轻松“云旅行”。[file:1]

## 功能特性

- 随机风景图片展示：页面从图片服务（如 Lorem Picsum）拉取高清风景图，每次刷新都有新图。[file:1]
- 参数设置面板：提供多种可调参数，用于模拟不同使用环境（如是否使用润滑剂、中途鼓励提示开关、使用经验等级等）。[file:1]
- 智能计算与会话控制：支持“开始会话”“暂停”“停止”，并实时统计运行时间、完成进度、插入次数、剩余时间等指标。[file:1]
- 历史与日志记录：包含最近设置记录、会话历史、活动日志等区域，可扩展为本地存储或后端日志展示。[file:1]

## 技术栈

- 前端页面：HTML + CSS + JavaScript，无需后端即可本地打开或部署到 GitHub Pages。[file:1]
- 使用浏览器原生表单控件和按钮组件，实现参数输入与交互操作。
- 对接随机图片服务（如 Lorem Picsum）以动态加载风景图片资源。[file:1]

## 项目结构

- `index.html`：主页面，包含风景展示区、参数设置面板、统计与日志界面。[file:1]
- `styles/`：样式文件目录（例如 `style.css`），用于控制布局和视觉效果。
- `scripts/`：脚本文件目录（例如 `main.js`），负责参数逻辑、计时和界面更新。

## 使用方式

1. 下载或克隆本项目代码到本地。
2. 直接用浏览器打开 `index.html` 即可体验。
3. 如需部署到 GitHub Pages，可在仓库设置中开启 Pages，选择主分支保存即可。

## 使用场景

- 作为前端练手项目，用于练习表单交互、计时器、状态管理等基础能力。[file:1]
- 作为 UI/UX 原型，用来展示“参数设置 + 实时统计 + 日志记录”的完整交互流程。
- 作为随机风景图小玩具，在日常工作学习间隙用来放松心情。
