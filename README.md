## 🎯 详细功能说明

### 参数设置深度解析

#### 润滑剂使用判断
- **已使用**：
  - 系统会降低难度系数
  - 建议时间缩短 15-20%
  - 安全提示减少
  
- **未使用**：
  - 增加准备时间建议
  - 提升安全警告频率
  - 推荐更保守的参数

#### 中途鼓励提示机制
- **触发条件**：
  - 进度达到 25%、50%、75% 时自动触发
  - 用户暂停后恢复时显示
  - 剩余时间少于 10% 时加强鼓励
  
- **提示内容**：
  - 正向激励语句
  - 进度确认信息
  - 剩余时间提醒
  - 安全操作建议

#### 长度和直径影响
- **长度参数**：
  - 直接影响总时长计算
  - 每增加 1cm，时间增加约 5-8%
  - 初学者建议 ≤ 15cm
  - 专家级可适配 30cm+

- **直径参数**：
  - 影响难度系数和安全评估
  - 小直径（< 3cm）：快速模式
  - 中直径（3-5cm）：标准模式
  - 大直径（> 5cm）：谨慎模式，增加准备时间

#### 使用次数和经验联动
系统会综合使用次数和经验等级进行智能判断：

| 经验等级 | 建议次数范围 | 时间系数 | 安全提示 |
|----------|--------------|----------|----------|
| 初学者 | 1-10 次 | 1.5x | 高频率 |
| 中级者 | 10-50 次 | 1.2x | 中等频率 |
| 高级者 | 50-200 次 | 1.0x | 低频率 |
| 专家级 | 200+ 次 | 0.8x | 按需提示 |

### 实时统计算法

#### 运行时间计算
```
运行时间 = 当前时间戳 - 会话开始时间戳 - 暂停累计时长
```

#### 完成进度算法
```
完成进度 = (已用时间 / 预计总时长) × 100%
预计总时长 = 基础时长 × 长度系数 × 直径系数 × 经验系数 × 润滑剂系数
```

#### 剩余时间预测
```
剩余时间 = 预计总时长 × (1 - 完成进度)
动态调整：根据实时速度自动修正预测
```

## 📖 使用指南

### 快速开始流程

#### 第一步：选择模式
1. 打开应用，默认显示美景欣赏界面
2. 点击界面进入参数设置模式

#### 第二步：配置参数
1. **基础选项**：
   - 选择是否使用润滑剂
   - 开启/关闭中途鼓励提示

2. **尺寸参数**：
   - 输入玩具长度（建议初学者从 10-15cm 开始）
   - 输入玩具直径（建议从 2-3cm 开始）

3. **经验设置**：
   - 如实选择您的经验等级
   - 输入历史使用次数（可选）

#### 第三步：智能计算
- 点击「🧮 智能计算」按钮
- 系统自动分析参数并生成建议
- 查看预计时长和安全提示

#### 第四步：开始会话
- 点击「▶️ 开始会话」启动
- 实时统计面板开始更新数据
- 关注进度条和剩余时间

#### 第五步：会话管理
- 需要休息时点击「⏸️ 暂停」
- 继续时再次点击恢复
- 完成或需要停止时点击「⏹️ 停止」

### 高级使用技巧

#### 数据分析
- 定期查看「会话历史」了解使用趋势
- 分析平均完成时间，优化参数设置
- 根据经验提升逐步调整难度

#### 快速恢复
- 使用「最近设置记录」快速恢复常用配置
- 为不同场景保存多套参数方案

#### 日志监控
- 关注活动日志中的警告信息
- 根据日志优化使用策略

## 📊 数据安全与隐私

### 数据存储
- **存储位置**：浏览器 LocalStorage（仅本地）
